This project was born from a university database assignment and evolved into a full-featured public web app. Built to make complex SQL queries more accessible, it invites users to explore a structured view of exoplanets, stars, and published research.

Project Overview

The Exoplanet Explorer System was developed to help users navigate and interact with a detailed astronomical database. Initially created for a university project, I later expanded and refined it into a publicly usable web application.

The system supports SQL-based operations such as projections, joins, filters, and aggregations.
