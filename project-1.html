Blackjack Web App

A full-stack web application that brings the classic card game to life with modern features like betting, insurance, splitting, and session tracking. Built with React and Flask.

Live Link

Project Overview

This Blackjack game began as a passion project rooted in childhood memories of playing with friends. It evolved into a full-stack application designed to replicate and enhance the traditional experience, featuring advanced mechanics like split hands, dynamic betting with chips, insurance, and real-time session tracking.

Built using React for the frontend and Flask for the backend, the app leverages clean API integrations to manage gameplay, logic, and persistent state. I also added animations, audio feedback, and custom UI design to make the game feel polished, intuitive, and immersive.

The project served as a playground for refining my skills in asynchronous state management, RESTful API design, and interactive UI/UX development. It also gave me the oppurtunity to get more familliar with Python and React in a fun setting.
