Eternal Gauntlet

A nostalgic dive into classic turn-based RPGs, Eternal Gauntlet is a personal homage to dungeon-crawler games I loved growing up. Built in collaboration with a 5-person team, it was a rewarding challenge that pushed our systems and graphics programming skills.

Code Link

Project Overview

Eternal Gauntlet is a 2D turn-based dungeon crawler inspired by classic RPGs. Developed in C++ with OpenGL, the game includes enemy AI, multi-floor progression, and a custom combat system.

I worked alongside four other team members to bring this game to life, contributing heavily to core gameplay logic, combat mechanics, and rendering routines. We emphasized polished visuals and strategic gameplay variety.

This project gave me practical experience with C++ game architecture, graphics pipelines, and real-time debugging across a shared codebase.
